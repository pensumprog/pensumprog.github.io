Indhold programmering B

  • Stepwise Improvement: Lav lille ændring, test ændring, ret koden hvis den ikke virker, gentag
  • 4 clean code regler: god navngivning, ingen gentagelser, simpelt, ryd altid op i koden efter en ændring
  • Print til konsol
  • Anvend debugger
  • skriv unit test (ikke pensum)
  • Metoder/Funktioner: Setup, draw, mousePressed, keyPressed, size, background, clear, frameRate, mouseX, random
  • Variabler: key, mousePressed, mouseX, mouseY, frameCount.
